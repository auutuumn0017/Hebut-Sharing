> 夏夜蝉鸣，在整理保研资料时回忆起这三年时光，我看到太多鲜活的面孔，被囚禁在“保研”这个庞大的命题里。从课程作业、实验报告、期末考试，每一个与绩点挂钩的事物，都在无形中拉扯着我们的神经。

> 希望大家能够合理利用，减少患得患失，多些闲暇去看看傍晚绚烂的晚霞，去真正体验生活该有的万千感受。🌅

## 📚 仓库内容

本项目主要收录人工智能数数据科学学院本科阶段的相关课程资料，按学期与课程分类。每个课程目录下大致包含：

*   **实验与课设**：历次实验要求、实验报告参考、代码实现。
*   **期末复习资料**：知识点提纲、整理的复习笔记、往年回忆版考题。
*   **课程大作业**：完整项目工程、答辩 PPT 参考、说明文档。

## ⚠️ 食用指南与免责声明

1.  **合理利用，拒绝抄袭**：本仓库的初衷是打破信息壁垒，帮助大家理清学习思路、参考优秀范例，**绝非鼓励直接抄袭**。请在独立思考的基础上参考他人代码与报告，否则不仅可能导致查重问题，更会失去课程本身的锻炼意义。
2.  **资料时效性**：课程大纲、实验要求及考试重点可能随学年发生变化，请务必以当年老师的最新要求为准。
3.  **版权声明**：本仓库内由个人整理的笔记、代码等资料开源，但严禁用于商业用途。

## 🤝 参与贡献

众人拾柴火焰高。如果你手中也有优秀的复习笔记、完善的实验报告，或者发现仓库中的资料有误，非常欢迎你参与贡献！

*   **方式一**：通过提 Pull Request (PR) 将你的资料补充到对应目录。
*   **方式二**：在 Issues 中提出建议、补充往年考题回忆，或是反馈资料错误。
*   *(如果你对 Git 操作不熟悉，也可以直接联系仓库维护者进行资料投稿)*
